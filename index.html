Git ir Github

Git yra kodo versijavimo programa

GitHub svetainė, kurioje keliamos kodo versijos.

  1. Įsirašyti Git
  2. patikrinti ar įrašytas terminale git --v
  3. susikurti github
  4. cinfig'inti git'ą
    1. git config --global user.name ligitaZA
    2. git config --global user.email ligitazaicevaite@gmail.com
    3. git config --global core.editor code
    4. git config --global core.autocrlf true
    5. git config --global -e
  5. Patikrinti ar tvarkingas config failas - git config --global --edit
  6. Publish to Github

Terminalo komandos

  • ls / dir - rodo kuriame faile/aplankale esi. ls -a - parodys ir užslėptą failą
  • cd - changeDirectory - keičia direkoriją, leidžia vaikščioti tarp aplankalų
  • tab - skirtas užbaigti / užpildyti terminalo komandą / pateikti pasiūlymus
  • vidurinis / dešinys pelės mygtukas - įklijuoja tekstą i komandinę eilutę
  • arrow up/down - vaikščiojimas per terminalo komandų istoriją

Git Komandos

  • git init - inicializuoja Git repozitoriją
  • git status - parodo git repozitorijos statusą
  • git add ./*/- pridėti failus prie commitinimui paruoštų failų
  • git commit -m "zinute" - sukuriama kodo versija su žinute
  • git push - nusiunčia naujus commitus į serverį
  • git checkout commitID - nukeliauja į nurodytą kodo versiją
